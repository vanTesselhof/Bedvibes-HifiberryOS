usageData

Anonymous usage data

When enabled, the system will submit anonymised usage data to HiFiBerry. This helps us understand how the product is used and to better target our development efforts on the right features.

What's shared?

Anonymous data about which features and sources are used. No music listening data is shared.

To see the data that has been collected, log in to the product via SSH and run command /opt/hifiberry/bin/report-dump. Enable SSH in General > Remote Login.
